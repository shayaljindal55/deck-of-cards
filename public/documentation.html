Deck of Cards

Tech Stack Used:

Static web application built using HTML, CSS and React JS and hosted on AWS S3.

Some key features:

  1. Responsive
  2. Can deal/draw a specified number of cards 
  3. Can shuffle the deck
  4. Can sort the drawn cards
  5. Can save local game & resume later from the previous state
  6. Can reset the game (Consider when you seem to loose the game )
  7. Tooltips, Toastbox (error messages)

Unit Testing:

 The app covers unit testing of one React
component, and one pure JS function (sort function).

  • To run unit testing on local, use command `npm test`

Steps to run the app on localhost :

  • Download and install Node JS on your system. Follow this link to download the executable as per your OS.
  • Code the code from using command "git clone https://github.com/shayaljindal55/deck-of-cards.git"
  • Open the cloned code in any IDE (I prefer Visual Studio Code)
  • Run `npm install` to install all node dependencies
  • Open terminal and change the directory to the location where you have cloned the project (from step 2)
  • Run npm start. 
  • Hit http://localhost:3000/ in the browser of your choice.

Or skip the hustle and access the direct link to the app!
